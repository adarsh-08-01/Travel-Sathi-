## 📱 About

Travel Sathi is my **first iOS application**, built using **SwiftUI + Antigravity + Claude AI**.

The goal of this project is to create a smart travel companion that understands:

- Who you are
- Where you want to travel
- Your budget

Based on these inputs, the app generates personalized travel recommendations and trip plans.

Different travelers have different needs. A student and a business owner should not receive the same travel suggestions. Travel Sathi solves this by creating recommendations based on user profiles and budget preferences.
